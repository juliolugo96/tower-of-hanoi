# Hanoi Tower

Hanoi Tower video game created using QtCreator

## Description

In order to play, follow the steps below:

  - Go to folder `Hanoi32` or `Hanoi64` depending on your computer arch.
  - Run `./TowerOfHanoi`
  - If you wish to observe CPU playing, in the GUI select `Play` and then choose
  `Autoplay`.
  - If you wish to put your disks in arbitrary positions, please select `Game` 
  and then choose `New Arbitrary` and fill with your desired configuration.
